# SA Cheats — GTA San Andreas

Fast, single-file cheat code reference for GTA San Andreas. PS4/PS5 and Xbox layouts.

**[▶ Live Demo](https://zengm5057-sudo.github.io/sa-cheats)**

---

## Features

- Platform switcher — PlayStation ↔ Xbox button layouts
- Real-time search by name or button (e.g. R1, Circle)
- Category filters: Player · Weapons · Vehicles · Spawn · World · Peds · Time/Weather
- Favorites with localStorage persistence + clipboard export
- Grid / List view toggle · Sort A→Z · Random cheat
- Dark / Light theme · One-click copy · Toast notifications
- Keyboard shortcuts: `/` focus search · `Esc` clear
- Zero dependencies — pure HTML + CSS + JS

## Deploy to GitHub Pages

```bash
git init
git remote add origin https://github.com/zengm5057-sudo/sa-cheats.git
git add index.html README.md
git commit -m "feat: initial release"
git branch -M main
git push -u origin main
```

Then go to **Settings → Pages → Deploy from branch → main / root → Save**.

Live at: `https://zengm5057-sudo.github.io/sa-cheats`
