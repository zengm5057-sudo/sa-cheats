# 🎮 SA Cheats — GTA San Andreas Cheat Code Reference

A clean, fast, single-file cheat code reference for **GTA San Andreas** — built for PS4/PS5 and Xbox. No frameworks, no dependencies, just one HTML file.

**[▶ Live Demo](https://ahmed-rade.github.io/sa-cheats)**

---

## ✨ Features

| Feature | Details |
|---|---|
| **Platform Switcher** | Toggle between PlayStation and Xbox button layouts instantly |
| **Search** | Real-time search by cheat name or button (e.g. "R1", "Circle") |
| **Category Filters** | Player · Weapons · Vehicles · Spawn · World · Peds · Time/Weather |
| **Favorites** | Star cheats and they persist across sessions via `localStorage` |
| **Export Favorites** | Copy all saved cheats to clipboard in a clean text format |
| **Random Cheat** | Discover a random cheat with one click |
| **Grid / List View** | Toggle between card grid and compact list layout |
| **Sort A→Z** | Alphabetically sort all visible cheats |
| **Dark / Light Mode** | Full theme toggle, clean on both |
| **Keyboard Shortcuts** | `/` to jump to search · `Esc` to clear |
| **Copy to Clipboard** | One-click copy per cheat with toast confirmation |
| **Toast Notifications** | Non-intrusive feedback for all actions |
| **Scroll to Top** | Quick-return button appears on scroll |
| **Responsive** | Works on mobile, tablet, desktop |
| **Zero Dependencies** | Pure HTML + CSS + JS · No libraries · No build step |

## 📋 Cheat Categories

- **Player** — Health, armor, money, wanted level, stats
- **Weapons** — Weapon sets, infinite ammo, hitman rank
- **Vehicles** — Traffic behavior, flying cars, handling
- **Spawn** — Vehicles and items (Tank, Hydra, Jetpack…)
- **World** — Gameplay modifiers (Bobble Head, Carnival…)
- **Peds** — Pedestrian behavior, riot mode, gang members
- **Time/Weather** — Rain, fog, sunny, sandstorm, time of day

## 🚀 Deploy to GitHub Pages

**Step 1 — Create the repo**
```bash
git init
git remote add origin https://github.com/Ahmed-Rade/sa-cheats.git
```

**Step 2 — Add and commit**
```bash
git add index.html README.md
git commit -m "feat: initial release - SA Cheats v1.0"
git branch -M main
git push -u origin main
```

**Step 3 — Enable GitHub Pages**
- Go to repo **Settings → Pages**
- Source: `Deploy from branch`
- Branch: `main` / root
- Click **Save**

Your site will be live at: `https://ahmed-rade.github.io/sa-cheats`
